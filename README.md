# HERO.IO - App Showcase Platform

## Description

HERO.IO is a modern React-based web application where users can explore and discover different productive applications. Users can view apps, see detailed information, and install/uninstall apps using a simple interface.

🔗 **Live Link:** [https://your-live-link.com](https://your-live-link.com)
🛠️ **Technology Stack:** React, React Router, Context API, Tailwind CSS, DaisyUI, React Icons, React Toastify, React Spinners

---

## Overview

HERO.IO is an app marketplace-style UI built with React. It allows users to browse apps, view detailed pages, and manage an installed apps list using Context API.

This project focuses on practicing modern React concepts such as:

* Context API for global state management
* Custom Hooks
* Dynamic Routing (React Router)
* Conditional Rendering
* Component-based architecture

## Features

* 🏠 Modern and fully responsive homepage with hero section
* 📱 Trending apps section with dynamic data rendering
* 📄 Individual app details page with full information
* ⚡ Install / Uninstall system using Context API
* 🔔 Toast notifications for user actions
* 🔄 Loading spinner during API/data fetching
* 📊 Stats section with animated data display
* 🧭 Responsive navigation bar with active link highlighting
* ❌ Custom 404 Not Found page

## How to Run Locally

### 1. Clone the repository

```bash
git clone https://github.com/your-username/hero-io.git
```

### 2. Navigate to project folder

```bash
cd hero-io
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start development server

```bash
npm run dev
```

### 5. Open in browser

```text
http://localhost:5173
```
